examples/: folder contains BotDetect Java Captcha library code examples and demos. Deploy .war files to your servlet container.

javadoc/: folder contains javadoc documentation of BotDetect Java Captcha library.

pronunciations/: folder contains the default pronunciation sound packages.

changelog.txt: file contains the version history of BotDetect Java Captcha library releases.

botdetect-license.html: file contains the licensing terms for using BotDetect Java Captcha library.
